Detailed Description

Class SIDhandler(ServerInitiatedDisconnect-Handler)

<alexej date="21.11.2008"> This class is used for handling various ServerIniated disconnect requests. This is necessary, because this requests are blocking (they wait for answer-disconnect message). The problem is, that to get the answer - services should be in unblocked state. To get this problem solved, an own thread for all SID request is presented in Component. So all requests from all Ports are handled sequentially in SIDhandler class.
