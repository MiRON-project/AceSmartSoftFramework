Detailed Description

template<class P, class E, class S>
class SmartACE::EventServer< P, E, S >

Template for generating a class which handles the event service on server side.

Template parameters:

  • P: Activation parameter class (Communication Object)
  • E: Event answer class (Communication Object)
  • S: State class (provides current information needed in testEvent() to check whether an event activation fires or not)
